É possível mexer com os parâmetros de forma a obter a representação energética de diversos pares primário-secundário. Alguns parâmetros de massa interessantes são vistos abaixo

Terra-Lua: μ ≈ 0.01215;
Sol-Júpiter: 	μ ≈ 9.5368e-4;
Sol-Terra: μ ≈ 3.00348e-6;
Sol-Marte: μ ≈ 3.22658e-7;
Plutão-Caronte: μ ≈ 0.106.

O parâmetro de massa genérico μ ≈ 0.3 nos dá uma superfície de potencial efetivo com curvas bem delineadas e é um ponto de partida interessante para ver as características do PRTC.
